## Installation et Utilisation 🚀

1. Clonez le repository :

```bash
git clone https://github.com/NINJAxARASHI/MyPortfolio.git
```

2. Installez les dépendances :

```bash
npm install
```

3. Lancez le projet en mode développement :

```bash
npm start
```
